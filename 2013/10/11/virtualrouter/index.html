利用VirtualRouter创建WIFI热点

Oct 11th, 2013

VirtualRouter——自由、免费、开源的一款能够用来虚拟WiFi热点的软件

什么是VirtualRouter？

VirtualRouter是一款自由、免费、开源的虚拟路由器的软件，可以运行在Windows8、Windows7、Windows Server 2008 R2平台。通过VirtualRouter，用户可以通过无线的方式与其他WIFI设备（笔记本/手机/平板电脑）共享任何互联网连接（WIFI，局域网，ADSL等等）。这些设备可以通过类似WIFI热点的方式连接到VirtualRouter借以访问互联网，其建立的连接使用WPA2加密方式加密，十分安全。（WPA2是当前最安全的无线加密方式）

如何使用

  1. 首先当然是下载软件。下载地址
  2. 下载之后双击安装。



  3. 安装完之后VirtualRouter会自动打开，按照图示填写新建WIFI热点的信息，然后点击Start Virtual Router。

  4. 如果一切顺利，在无线连接的信息里面应该有类似下图的显示。

出错

如果遇到了“The group or resource is not in the correct state to perform the requested operation”错误，如下图。

请尝试用以下方式解决：

  1. 在设备管理器中启用“Microsoft Virtual WiFi Miniport Adapter”。
    控制面板–>系统和安全，在弹出的窗口中，找到系统一栏下面的设备管理器，在打开的窗口中找到网络适配器，打开子项目，在“Microsoft Virtual WiFi Miniport Adapter”单击右键，然后点击启用。



  2. 在管理员模式下启动cmd.exe，然后执行

具体步骤是，按下键盘上的Win键,输入cmd.exe,右键以管理员模式运行，在命令行中粘贴上面提到的命令，然后单击回车。



3. 选择1或2之中的一个方法，执行之后重新点击 “Start Virtual Router”。

结语

如果一切顺利，那么，现在你就可以在没有路由的情况下，将笔记本虚拟出一个WiFi热点供手机等其他的WiFi设备使用了。
